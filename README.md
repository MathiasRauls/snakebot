# snakebot
An AI trained to do nothing but play snake endlessly... 🐍

![snakebot screenshot](https://github.com/MathiasRauls/snakebot/blob/main/snakebot.png)

## Getting Started

### From GitHub (recommended)
```bash
pip install git+https://github.com/MathiasRauls/snakebot.git
```

### From source
```bash
git clone https://github.com/MathiasRauls/snakebot.git
cd snakebot
pip install -e .
```
___

## Usage
Run the game with:
```bash
snakebot
```
___

## Controls
**Movement**: <kbd>up</kbd> <kbd>down</kbd> <kbd>left</kbd> <kbd>right</kbd><br>
**Quit**: <kbd>q</kbd>
___

## Features
**Big Ole Sniffer**: This 'lil guy can smell the 'food' denoted by the red color growing more intense the closer the snake gets to its 'prey'.<br>
**Atmosphere**: The perfect blend of synthetic slavery, & retro arcade games.
